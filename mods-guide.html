üîß Mod Loader Installation - BepInEx Setup

A mod loader is required software that allows My Winter Car to recognize and run community-made mods. For MWC, BepInEx 5 is the primary and most widely supported mod loader.

Why BepInEx?

  • ‚úÖ Most compatible: 180+ mods on Nexus require BepInEx
  • ‚úÖ Actively maintained: Stable version 5.4.23.4 released and tested
  • ‚úÖ Community standard: Recommended in all official tutorials
  • ‚ùå MSCLoader does NOT work: The My Summer Car loader is incompatible with MWC (an updated version 1.4 is reportedly in development but not yet released)
üö´ Critical: BepInEx vs MelonLoader Conflict

You CANNOT run BepInEx and MelonLoader simultaneously. Installing both will cause conflicts and break mod functionality.

Recommendation: Use BepInEx only. Most MWC mods are designed for BepInEx, and MelonLoader has limited mod support as of January 2026.

BepInEx Installation Guide

Step 1: Download BepInEx

  1. Visit the BepInEx GitHub Releases page
  2. Scroll to Assets section
  3. CRITICAL: Download BepInEx_x64_5.4.23.4.zip (or latest x64 version)
  4. DO NOT download x86 version - it will not work with My Winter Car

Step 2: Locate Game Directory

  1. Open Steam library
  2. Right-click My Winter Car
  3. Select Manage ‚Üí Browse Local Files
  4. This opens your game folder (typically: C:\Program Files\Steam\steamapps\common\My Winter Car)

Step 3: Install BepInEx Files

  1. Extract the BepInEx_x64_5.4.23.4.zip file
  2. Copy ALL contents (4 files + 1 folder) into your My Winter Car game directory
  3. Files to copy: changelog.txt, doorstop_config.ini, winhttp.dll, plus the BepInEx folder

Step 4: Generate Configuration Files (CRITICAL)

  1. Launch My Winter Car through Steam
  2. The game will attempt to start but likely crash or close immediately - This is normal!
  3. This failure is intentional - it forces BepInEx to create necessary folders
  4. Close the game (if it didn't auto-close)
  5. Check your game directory - you should now see a populated BepInEx folder with subfolders including plugins
üî¥ CRITICAL STEP - Config File Edit

Without this edit, BepInEx will NOT work!

Step 5: Edit BepInEx Configuration (MANDATORY)

  1. Navigate to your game directory
  2. Open the BepInEx folder
  3. Open the config subfolder
  4. Find BepInEx.cfg file
  5. Right-click ‚Üí Open With ‚Üí Notepad (or any text editor)
  6. Scroll to the bottom of the file
  7. Find the section: [Preloader.Entrypoint]
  8. Locate the line: Type = Application
  9. Change it to: Type = MonoBehaviour
  10. Save the file and close
‚úÖ Verify BepInEx Installation

Launch My Winter Car. If BepInEx is working correctly, you will see:

  • A console/command prompt window appears alongside the game
  • Console shows messages like: [Info : BepInEx] BepInEx 5.4.23.4 - My Winter Car
  • Messages about "loaded plugins" or "registered keybinds"

If you see these, BepInEx is successfully installed!

Installing Your First Mod

  1. Download a mod from Nexus Mods (usually a .dll file or a zip containing one)
  2. Extract the mod if it's in a zip file
  3. Navigate to: My Winter Car\BepInEx\plugins\
  4. Copy the mod's .dll file into the plugins folder
  5. Launch the game - the mod will load automatically
  6. Most mods use F1, F5, or K to open their in-game menu
üí° Linux / Steam Deck Users

BepInEx works on Linux via Proton with a launch option workaround:

  1. Right-click My Winter Car in Steam ‚Üí Properties
  2. In Launch Options, paste: WINEDLLOVERRIDES="winhttp=n,b" %command%
  3. Follow normal BepInEx installation steps

See BepInEx Proton Documentation for details.

‚ö†Ô∏è HIGH RISK - Backup Required

üîß Troubleshooting Common Modding Issues

BepInEx Not Loading

Problem: Game launches but no console window appears

Solution 1: Wrong BepInEx version downloaded

  • Delete all BepInEx files from game directory
  • Re-download x64 version (NOT x86)
  • Re-install following instructions above

Solution 2: Config file not edited

  • Verify BepInEx.cfg shows Type = MonoBehaviour
  • Check you edited the file in BepInEx\config\, not the main folder

Mod Not Working After Installation

Diagnosis Checklist
  1. Is BepInEx loaded? Console window should appear on game launch
  2. Correct folder? Mods must be in BepInEx\plugins\, not root game folder
  3. File type correct? Should be .dll file, not .zip
  4. Mod conflict? Disable other mods one-by-one to test
  5. Game version mismatch? Check mod page for compatibility notes
  6. Read console output: Error messages in console window reveal issues

Game Crashes After Installing Mods

Step-by-step debugging:

  1. Remove all mods from plugins folder
  2. Launch game to verify it works without mods
  3. Add mods back ONE AT A TIME
  4. Launch game after each mod
  5. Identify which mod causes the crash
  6. Check that mod's Nexus page for updates or bug reports

Save File Corruption

Symptom: Game won't load save / Error messages

Solution:

  1. Restore from backup (this is why backups are critical!)
  2. If no backup exists, try loading an autosave
  3. As last resort, use save editor to repair file (risky)

BepInEx and MelonLoader Conflict

Problem: Mods not loading / console errors

Cause: Both frameworks installed simultaneously

Solution:

  1. Choose ONE framework (recommend BepInEx for MWC)
  2. Completely remove the other:
    • Delete MelonLoader files (MelonLoader folder, version.dll)
    • OR delete BepInEx files
  3. Reinstall chosen framework cleanly

üìã Best Practices & Safety Guidelines

‚úÖ Golden Rules of Modding
  1. ALWAYS backup saves before modding - Cannot be overstated
  2. Read mod descriptions completely - Many issues come from skipping instructions
  3. Use one mod loader only - BepInEx recommended for MWC
  4. Install mods one at a time - Makes troubleshooting easier
  5. Check compatibility before updating game - Updates break mods
  6. Keep mods updated - Check Nexus after each game patch
  7. Don't use MSC mods in MWC - Different games, different code
  8. Scan downloads for viruses - Nexus is safe, but always verify

Recommended First-Time Setup

Conservative modding for new players:

  1. Play vanilla for 2-3 hours to understand mechanics
  2. Install only BepInEx (no mods yet)
  3. Add SAW - Save AnyWhere first (prevents progress loss)
  4. Add ShowBoltSize when you start Rivett assembly
  5. Add My Winter Backpack when inventory management becomes tedious
  6. Only add cheat mods if truly frustrated (consider if challenge is the point)
