2. Getting Started: Acquiring Parts

Parts for your engine can be acquired from multiple sources. Plan your purchases carefully to minimize trips and waiting time.

Classifieds Magazine (Used Parts)

üì∞ How to Order from Classifieds Magazine
  1. Buy the Classifieds Magazine from the grocery rack at PSK Gas Station for 15 MK
  2. Return to your apartment and use the phone
  3. Open the magazine and dial the phone numbers for desired parts using the numpad
  4. Important: Type the number correctly and do not hang up until the NPC ends the call
  5. After approximately 2 days, you'll receive a post-order notice in your mailbox
  6. Pick up your order at PSK Gas Station after 10:00 PM and pay the cashier

Note: The Classifieds Magazine resets weekly, so new parts become available regularly. Parts are in used condition ‚Äî quality varies.

4. Step-by-Step Engine Assembly

Follow this sequence for successful engine assembly. While some flexibility exists, maintaining this general order helps avoid common mistakes.

Phase 1: Engine Block Foundation

1Build the Engine Block
  1. Start with the engine block mounted on the engine stand
  2. Install the crankshaft into the block
  3. Install main bearings (if required separately)
  4. Attach connecting rods
  5. Install pistons onto the connecting rods
  6. Mount the oil pump
  7. Attach the oil pan

Phase 2: Cylinder Head Installation

2Install the Cylinder Head
  1. Place the head gasket on the engine block
  2. Lower the cylinder head onto the block
  3. Install the camshaft
  4. Attach rocker arms
  5. Install the rocker cover (after valve adjustment, if needed)

Phase 3: Fuel & Ignition System

3Install Fuel and Ignition Components
  1. Install the fuel pump (mechanical type)
  2. Mount the carburetor
  3. Install spark plugs
  4. Attach the distributor

Phase 4: Cooling System

4Install Cooling Components
  1. Install the thermostat
  2. Mount the water pump (if not done during timing setup)
  3. Attach the fan

Phase 5: Electrical Components

5Install Electrical System
  1. Install the alternator
  2. Attach the fan belt
  3. Mount the starter motor

Phase 6: Transmission Connection

6Connect to Transmission
  1. Attach the rear plate
  2. Install the flywheel (or flexplate for automatic)
  3. Install clutch components in order:
    • Clutch Disk
    • Clutch Cover Plate
    • Clutch Pressure Plate
üö® No Rev Limiter Warning

The Corris Rivett has no electronic rev limiter. Over-revving will cause catastrophic engine damage. Always watch your tachometer!

5. Timing Belt Installation

The timing belt system is critical ‚Äî incorrect timing will prevent the engine from running. Pay close attention to this section.

‚öôÔ∏è Timing Belt Assembly Sequence
  1. Install the auxiliary shaft
  2. Attach the auxiliary shaft sprocket
  3. Install the camshaft sprocket
  4. Important: Before installing the timing belt, remove:
    • Crankshaft Pulley
    • Water Pump / Water Pump Pulley
  5. Install the timing belt over:
    • Crankshaft
    • Auxiliary Shaft Sprocket
    • Camshaft Sprocket
  6. Reinstall the water pump and pulley
  7. Reinstall the crankshaft pulley
  8. Attach the timing belt cover (uses 2x 6mm bolts to cylinder head)
‚ö†Ô∏è Critical: Timing Alignment

Proper timing belt alignment is crucial. The timing belt synchronizes the crankshaft, auxiliary shaft, and camshaft. If the engine cranks but won't start, timing may be off ‚Äî double-check that all sprockets are correctly positioned.

6. Pre-Start Checklist

Before attempting to start the engine for the first time, verify all of the following:

  • All bolts are properly tightened
  • Engine oil is filled to correct level
  • Coolant is filled
  • Brake fluid is topped up
  • Fuel tank has gasoline
  • Battery is fully charged
  • All electrical connections (battery, alternator, starter) are correct
  • Wiring is properly connected (see Wiring Guide)

If building in the garage at the parents' house, use the motor hoist to make engine installation much easier. The apartment does not have this equipment.

7. Cold Starting Procedure

The Corris Rivett uses a mechanical fuel pump and carburetor, requiring a specific cold start procedure:

‚ùÑÔ∏è Cold Start Steps
  1. Engage the choke fully (pull the choke knob all the way out)
  2. Crank the engine for approximately 10 seconds (real-time)
  3. Disengage the choke (push the knob back in)
  4. Wait 10-15 seconds
  5. Try starting again while giving it some gas
  6. Once running, you can re-engage the choke to warm up the engine
  7. Remove the choke when driving
üîß Why This Works

The mechanical fuel pump needs to build sufficient fuel line pressure, which takes a few cranks. The choke enriches the fuel-air mixture, which is necessary because cold fuel vaporizes less efficiently.

‚ö†Ô∏è Winter Warning

In extreme cold (below -15¬∞C), the battery may be too weak to crank. Consider using a block heater to warm the engine, or bring the battery indoors to warm up before starting.

8. Troubleshooting: Engine Won't Start

If your engine won't start after assembly, check these common issues:

Timing Problems

üî¥ Engine Cranks But Won't Fire

Symptom: Engine turns over but never catches

Likely Cause: Incorrect timing belt alignment

Solution:

  • Verify timing belt is properly installed over all three sprockets
  • Check that sprockets are correctly positioned
  • Remove timing belt cover and visually inspect alignment

Fuel System Issues

üî¥ No Fuel Reaching Carburetor

Symptom: Engine cranks, no fuel smell

Solution:

  • Verify fuel pump is installed correctly
  • Check fuel line connections
  • Ensure fuel tank has gasoline
  • Try extended cranking with choke engaged (builds fuel pressure)

Electrical Faults

üî¥ No Cranking or Clicking Sounds

Symptom: Nothing happens when turning key

Solution:

  • Check battery charge and terminal connections
  • Verify starter motor wiring
  • Check alternator connections
  • Inspect ground cables for corrosion
