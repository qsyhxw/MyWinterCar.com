üö® CRITICAL: Backup Your Save BEFORE Editing

Save editors can permanently corrupt your progress. These tools were designed for My Summer Car and have partial compatibility with My Winter Car. Even with updated MWCEditor versions, unexpected behavior can occur. Always create a backup copy of your entire save folder before making ANY changes. Losing 20+ hours of progress because you didn't spend 30 seconds copying a folder is a terrible way to learn this lesson.

Backup Location: Copy entire folder at %USERPROFILE%\AppData\LocalLow\Amistech\My Winter Car to a safe location (Desktop, Documents, external drive).

üîß Which Save Editor Should You Use?

There are currently two save editors available for My Winter Car, each with different strengths and limitations. Both are community projects adapted from My Summer Car tools.

üì• Installation & Setup Guide

Prerequisites & Requirements

üíª System Requirements

  • Windows 7/8/10/11 (64-bit)
  • x64 version of editor required
  • No additional dependencies
  • ~5 MB disk space

üéÆ Game Requirements

  • My Winter Car installed
  • At least one save file created
  • Game closed before editing
  • Save folder permissions

üìÅ Save File Location

  • %USERPROFILE%\AppData\LocalLow\Amistech\My Winter Car
  • Main file: savefile.txt
  • Backup regularly
  • Editor auto-detects folder

Step-by-Step Installation

1

Create Backup

CRITICAL FIRST STEP: Before downloading anything, backup your save:

  1. Press Windows + R to open Run dialog
  2. Type %USERPROFILE%\AppData\LocalLow\Amistech and press Enter
  3. Right-click "My Winter Car" folder ‚Üí Copy
  4. Paste to safe location (Desktop, Documents, external drive)
  5. Rename copy to "My Winter Car BACKUP [DATE]"
2

Download MWCEditor

Choose one download source:

Option A - GitHub (Recommended):

  1. Visit github.com/Marcinosky/MWCEditor/releases
  2. Download latest .zip file under "Assets"
  3. Right-click downloaded file ‚Üí Extract All

Option B - Nexus Mods:

  1. Visit nexusmods.com/mywintercar/mods/246
  2. Click "Files" tab ‚Üí Manual Download
  3. Extract downloaded archive
3

Launch MWCEditor

No installation required - it's a standalone executable:

  1. Navigate to extracted folder
  2. Find MWCEditor.exe (or MSCEditor.exe for x64 version)
  3. Right-click ‚Üí Run as Administrator (recommended)
  4. If Windows SmartScreen appears, click "More info" ‚Üí "Run anyway"

‚ö†Ô∏è Make sure you're using the x64 (64-bit) version - 32-bit version does NOT work with My Winter Car.

4

Open Your Save File

MWCEditor should auto-detect your save folder on first launch. If not:

  1. Click File ‚Üí Open in editor menu bar
  2. Navigate to %USERPROFILE%\AppData\LocalLow\Amistech\My Winter Car
  3. Select savefile.txt
  4. Click "Open" - editor will load your save data

‚úì Success indicator: You'll see your player name, money amount, and vehicle data populate in the editor interface.

5

Make Changes & Save

Edit your save data carefully:

  1. Make desired changes (see sections below for specific edits)
  2. Click File ‚Üí Save (or Ctrl+S)
  3. Close the editor completely
  4. Launch My Winter Car to test changes

üö® NEVER edit while game is running - this WILL corrupt your save. Always close game before opening editor.

‚úÖ Verification Checklist

  • ‚òëÔ∏è Backup created and stored safely outside game folder
  • ‚òëÔ∏è Using x64 version of editor (not 32-bit)
  • ‚òëÔ∏è Game is completely closed before editing
  • ‚òëÔ∏è Editor successfully opened savefile.txt
  • ‚òëÔ∏è Can see player data loaded in editor interface

üí∞ Common Save Edits & Features

1. Money Editing

ü™ô Infinite Money Cheat

The most popular save edit. Perfect for skipping the early-game money grind and focusing on building your Rivett.

How to Edit:

  1. In MWCEditor, find the "Money" or "Player" tab
  2. Locate "Money" or "Cash" field (displays current Markka balance)
  3. Enter desired amount (e.g., 999999 for ~1 million MK)
  4. Click "Save" ‚Üí Close editor ‚Üí Launch game

‚ö†Ô∏è Recommended Amounts: 50,000 MK = comfortable early game, 200,000 MK = all Rivett parts + safety buffer, 999,999 MK = effectively infinite money

2. Vehicle Teleportation

üöó Teleport Vehicles & Player

Rescue vehicles stuck in forests, underwater, or off-map locations. Essential for recovering from catastrophic failures.

How to Teleport:

  1. Open Tools menu in MWCEditor
  2. Select "Teleport" or "Position" tool
  3. Choose object to teleport (Player, Sorbet, Rivett, etc.)
  4. Select preset location:
    • "Garage Table" - Parents' house garage (safest)
    • "Player Apartment" - Your apartment in Per√§j√§rvi
    • "Fleetari" - Repair shop in Loppe
    • Or enter custom coordinates
  5. IMPORTANT: Set "Height Offset" to 1.0 meter

üö® Why Height Offset Matters: Without the +1m offset, vehicles spawn embedded in the ground, causing physics explosions that launch them into orbit or destroy them instantly. Always add vertical clearance.

3. Survival Stats Management

‚ùÑÔ∏è Edit Hunger, Thirst, Temperature, Problem

Modify all 6 survival stats to prevent death or skip tedious maintenance. Useful for testing or creative builds.

How to Edit Stats:

  1. In MWCEditor, navigate to "Player Stats" tab
  2. Find survival stat fields:
    • Hunger: Set to 0 (full) or 100 (starving)
    • Thirst: Set to 0 (hydrated) or 100 (dehydrated)
    • Temperature: Set to 37.0 (normal body temp ¬∞C)
    • Fatigue: Set to 0 (rested) or 100 (exhausted)
    • Urination: Set to 0 (empty bladder)
    • Problem (Alcoholism): Set to 0 (no addiction) - WARNING: Setting this too low removes beer requirement!
  3. Save changes and test in-game

‚ö†Ô∏è Problem Stat Caution: The "Problem" stat represents alcoholism. Setting it to 0 means you won't need beer to survive, which removes a core survival mechanic. Consider setting to 10-20 instead to keep beer requirement but reduce urgency.

4. Vehicle Diagnostics (Rivett Report)

üîç Rivett Report - Part Status Checker

MWCEditor's killer feature (added Jan 2026). Shows which Rivett parts are loose, damaged, or missing. Essential for diagnosing why your car won't start.

How to Use:

  1. Open MWCEditor with your save loaded
  2. Click "Rivett Report" or "Car Report" tab
  3. View table showing:
    • Part name
    • Installation status (installed/missing)
    • Bolt tightness (loose bolts highlighted)
    • Condition percentage
    • Damage indicators
  4. Identify problems ‚Üí Fix in-game or edit directly in editor

‚ö†Ô∏è Current Limitations: Wiring system not yet implemented in report. Electrical issues won't show up here. Developer is working on adding wiring diagnostics in future updates.

5. Time & Date Adjustment

‚è∞ Skip Time / Change Date

Fast-forward to specific times for job scheduling, skip nights, or reach weekend events like ice racing or Flea Market.

How to Edit Time:

  1. Find "Time" or "Date" section in editor
  2. Adjust hour, day, or week values
  3. Save and load game

üö® BROKEN IN MWCEditor: Time/weather editing currently has bugs. Values may not apply correctly or may reset. Developer is aware and working on fix. Use in-game bed sleep for time skipping instead (safer).

üéØ Advanced Features & Pro Tips

üîß Bolt Tightness Editor

Tighten all Rivett bolts to 100% without manually using wrenches. Prevents parts falling off during driving.

  • Open Rivett Report
  • Select "Tighten All Bolts"
  • Or manually set bolt values to 8 (max tightness)
  • Save before major drives

üó∫Ô∏è Coordinate Finder

Get exact X/Y/Z coordinates for custom teleport locations. Useful for creating custom spawn points.

  • Place character/vehicle at desired spot in-game
  • Save game
  • Open editor ‚Üí Tools ‚Üí Position
  • Copy coordinates for future use

üè™ Inventory Management

Add items directly to player inventory without shopping trips to PSK or Fleetari.

  • Navigate to "Inventory" or "Items" tab
  • Spawn items (beer, food, parts)
  • ‚ö†Ô∏è Feature untested in MWCEditor
  • High risk of corruption - backup first!

üìä Condition Editing

Restore part condition to 100% NOS quality. Useful after crashes or wear damage.

  • Open Rivett Report or part list
  • Find damaged part
  • Set "Condition" field to 100
  • Cheaper than buying NOS parts at Fleetari

üîì Unlock Achievements

Some MSC achievements may be editable. MWC achievement system unknown.

  • Explore "Achievements" tab if present
  • May not work for MWC (Steam achievements)
  • Not recommended - defeats purpose
  • Use only for testing

üö´ Reset Fines & Tickets

Clear police fines from speeding, expired inspection, or fuel tax violations.

  • Find "Police" or "Fines" section
  • Set fine amount to 0
  • Clear inspection violation flags
  • Useful after false police stops

üí° Pro Tips for Safe Editing

  • Make incremental changes: Edit one thing at a time, test in-game, then make next change. Don't modify 10 values at once.
  • Keep backup versions: Create numbered backups (Backup_v1, Backup_v2) so you can roll back to specific points.
  • Close game COMPLETELY: Check Task Manager to ensure MWC.exe isn't running before opening editor.
  • Don't spawn unknown items: Spawning items with incorrect IDs can crash the game or corrupt save. Stick to money/stat editing.
  • Height offset rule: Always add +1 meter when teleporting vehicles. Physics explosions are not fun.
  • Test after editing: Load save, check changes worked, play for 5-10 minutes, then save in-game before doing more edits.
  • Join community Discord: Ask questions before attempting risky edits. Community knows common pitfalls.

‚ö†Ô∏è Known Issues & Limitations

üîß MWCEditor Current Bugs (as of Jan 2026)

  • Rivett Wiring: Not implemented yet. Electrical issues can't be diagnosed or fixed in editor.
  • Weather System: Time/weather combo values are broken. Editing causes incorrect values or game crashes.
  • Item Spawning: Untested functionality. Item ID tables from MSC may not match MWC. High corruption risk.
  • Documentation: Help text and warnings are outdated from MSC version. Some tooltips reference wrong game.
  • Some Rivett Parts: A few specific parts may not appear in editor even with Rivett Report. Developer working on fixes.

üö´ MSCEditor Incompatibilities with MWC

  • Rivett Completely Invisible: Original MSCEditor can't see Rivett parts, attributes, or vehicle data at all.
  • No Rivett Report: Car diagnostics only work for MSC Satsuma, not MWC Rivett.
  • 32-bit Version: Doesn't work with MWC at all. Must use x64 version or it won't open save files.
  • MWC-Specific Features: Any new mechanics added to MWC that weren't in MSC will be missing from editor.
  • Higher Corruption Risk: Because it's not designed for MWC, unexpected data can cause silent corruption.

‚úÖ Pre-Edit Safety Checklist

  • ‚òëÔ∏è Backup created: Full copy of My Winter Car folder stored safely
  • ‚òëÔ∏è Game closed: MWC.exe not running in Task Manager
  • ‚òëÔ∏è Using MWCEditor: Not MSCEditor (unless specific reason)
  • ‚òëÔ∏è x64 version: Not 32-bit version
  • ‚òëÔ∏è Know what you're editing: Clear goal (money, teleport, etc.)
  • ‚òëÔ∏è Read limitations: Aware of what doesn't work (wiring, weather)

üìã Best Practices for Long-Term Safety

  • Backup rotation: Keep 3-5 backup versions at different progress points. Delete oldest when creating new.
  • Name backups clearly: "MWC_Backup_Day15_BeforeRivettEdit_Jan06" beats "Backup1".
  • External storage: Keep at least one backup on USB drive or cloud storage (OneDrive, Google Drive).
  • Test immediately: After editing, load save and play for 10+ minutes to catch corruption early.
  • Don't edit critical moments: Avoid editing during important builds or before race events. Wait for stable moment.
  • Document changes: Keep text file noting what you edited (e.g., "Added 50k MK on Jan 6"). Helps troubleshoot later issues.
  • Regular in-game saves: After editor changes work, make multiple in-game saves (toilet saves). Don't rely on single save point.
  • Community updates: Check GitHub/Nexus monthly for MWCEditor updates. Bug fixes released regularly.

üö® What NOT To Do

  • NEVER: Edit while game is running (instant corruption)
  • NEVER: Skip backups "just this once" (Murphy's Law applies)
  • NEVER: Edit save file directly in text editor (guaranteed corruption unless you're a developer)
  • NEVER: Spawn items with unknown IDs (crashes/corruption)
  • NEVER: Teleport to coordinates without testing (can spawn inside solid objects)
  • NEVER: Edit weather/time in current MWCEditor version (known broken)
  • NEVER: Overwrite your only backup (keep multiple versions)
  • NEVER: Ignore corruption signs (weird physics, missing items, crashes) - restore backup immediately

üì¶ Importing My Summer Car Saves to MWC

My Winter Car supports importing progress from My Summer Car, allowing you to carry over unlocks and some items. This is useful for players transitioning from MSC to MWC who want to skip re-grinding basic progress.

üéØ What Gets Imported

  • Vehicle Unlocks: Access to vehicles you owned in MSC
  • Some Items: Certain inventory items transfer
  • Player Stats: Some progression metrics
  • Money: Your MSC Markka balance

‚ö†Ô∏è What DOESN'T Import

  • Satsuma ‚Üí Rivett: Your built Satsuma doesn't become a Rivett. Rivett starts unbuilt.
  • Map-Specific Locations: Item positions won't match (different map layout)
  • Some Quest Progress: MWC has different quest system
  • Current Save State: Must start NEW game in MWC after import (can't continue mid-game)

Import Process

1

Locate MSC Save File

  1. Press Windows + R
  2. Type %USERPROFILE%\AppData\LocalLow\Amistech
  3. Open "My Summer Car" folder
  4. Find defaultES2File.txt
  5. Right-click ‚Üí Copy
2

Backup Current MWC Save

CRITICAL: Back up your current MWC progress before overwriting:

  1. Navigate to My Winter Car folder (same Amistech folder)
  2. Copy entire "My Winter Car" folder to safe location
  3. This ensures you can revert if import fails
3

Paste & Rename

  1. Open "My Winter Car" folder
  2. Right-click ‚Üí Paste (the defaultES2File.txt from MSC)
  3. Alternative method: Some users report needing to rename to savefile.txt instead
  4. If import doesn't work, try renaming defaultES2File.txt to savefile.txt
4

Start New MWC Game

  1. Launch My Winter Car
  2. Select "New Game" (not Continue)
  3. Game will import data from MSC save automatically
  4. Check inventory and vehicles to confirm import

‚ö†Ô∏è Will NOT work on existing MWC save - must start fresh game for import to trigger.

üí° Post-Import Editing

After importing MSC save, you can use MWCEditor to:

  • Adjust imported money amount (add more or reduce for challenge)
  • Teleport vehicles to preferred starting locations
  • Reset survival stats to fresh start values
  • Remove items you don't want transferred

Recommended: Use MSCEditor (original) for editing imported saves initially, as it understands MSC data structure better. After first few edits, switch to MWCEditor for ongoing modifications.
